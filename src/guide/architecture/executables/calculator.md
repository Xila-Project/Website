# 🧮 Calculator (executable)

The calculator executable provides a small expression-evaluation application with a WASM-focused UI path.

## Features

- Lexer, parser, and evaluator pipeline for arithmetic expressions.
- WASM UI integration through a dedicated interface module.
- Localization-aware fallback message when run outside intended targets.

## Target model

This executable is primarily intended for WASM environments.
