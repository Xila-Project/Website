# Graphical shell (executable)

The graphical shell executable is the desktop-style shell of Xila.

It manages login, desktop layout, shortcuts, and home/desk views while coordinating the active user session.

## Features

- Graphical login flow.
- Desktop layout and desk/home components.
- Optional on-screen keyboard mode via startup arguments.
- Session environment initialization after user login.
